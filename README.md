# 上下文分析

`上下文分析` 是一个面向 SillyTavern 的上下文观察与调试插件，主要用来帮用户看清楚：

- 当前 prompt 是由哪些部分组成的
- 哪一类内容最占上下文
- 世界书、角色卡、Persona、聊天记录分别贡献了多少
- 当前上下文是否有超预算、重复、冗余或结构分散的问题
- 结合你现有的插件生态，AI 更推荐先装哪个插件、先改哪一块

这个插件的定位不是“替代酒馆全部编辑器”，而是：

**把上下文看清楚，把问题指出来，把下一步该调什么说清楚。**

---

## 主要功能

### 1. 简单总览

用于快速查看当前上下文状态，包括：

- 当前接口类型
- 本轮上下文总量
- 上下文占比
- 世界书命中数量
- 最近快照时间
- 主要来源排行

适合先快速判断：

- 现在是不是聊天记录太长了
- 世界书是不是太重了
- 哪部分最占 token

### 2. 详细信息

详细页已经做成**分段切换**，不会把所有东西一股脑堆在一起。可以单独切换查看：

- 发送给 AI 的内容
- 详细来源
- 上下文占比
- 世界书状态
- 诊断结果
- 当前配置

其中世界书部分会区分：

- 角色卡主世界书
- 全局世界书启用集
- 本轮激活条目

### 3. AI优化建议

插件支持基于当前快照生成 AI 建议，不只是看占比，还会尝试回答：

- 当前最大问题是什么
- 先改聊天记录、世界书、预设，还是角色卡
- 哪类问题更适合直接用现有插件解决

AI 建议里会优先结合现有插件生态给出推荐，例如：

- 聊天记录太长：推荐记忆插件、自动总结扩展
- 世界书太重：推荐世界书扩展插件
- 预设太长：推荐预设管理器
- 角色信息分散：推荐角色卡自动更新扩展
- 格式、状态栏、正则修复：推荐正则管理器 / 自动修复格式插件、实时状态栏插件

### 4. 快照导出与复制

插件支持：

- 复制当前最终 prompt
- 导出当前快照

适合做：

- 问题复现
- 调试留档
- 对比不同配置前后的变化

### 5. 更新检查

插件设置页内置了版本与更新区域，可以：

- 查看当前版本
- 检查是否有新版本
- 发现新版本后直接更新

## 适合谁用

这个插件尤其适合以下用户：

- 世界书、预设、扩展比较多，已经分不清是谁在占上下文
- 角色卡、世界书、扩展注入混在一起，想先看清结构
- 想知道当前问题更适合“手动调整”还是“直接装插件解决”
- 想做 prompt 体检、上下文调优、问题定位
